Код решения

    
let car = {
    brand: "Toyota",
    model: "Camry",
    year: 2015
};

car.year = new Date().getFullYear();
car.color = "Чёрный";
delete car.model;

console.log("Свойство 'brand' существует:", "brand" in car);

for (let key in car) {
    console.log(key + ": " + car[key]);
}

let carCopy = { ...car };
carCopy.color = "Красный";

car.owner = {
    name: "Иван",
    age: 35
};

let { name } = car.owner;
console.log("Имя владельца:", name);

let keys = Object.keys(car);
let values = Object.values(car);
console.log("Ключи:", keys);
console.log("Значения:", values);
    


    
Вопросы и ответы

    

        

            
Как можно получить значение свойства объекта через точечную нотацию?

            
Через точку: object.property

        

        

            
В каких случаях удобнее использовать квадратные скобки для доступа к свойствам объекта?

            
Когда имя свойства хранится в переменной или содержит пробелы/спецсимволы.

        

        

            
Как можно изменить значение существующего свойства объекта?

            
object.property = новое_значение;

        

        

            
Как добавить новое свойство в объект?

            
object.newProperty = значение;

        

        

            
Как удалить свойство из объекта?

            
delete object.property;
